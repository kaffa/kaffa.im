问题现象
==========

在 Windows 上，通过音频线连接 Bose SoundLink，但声音会自动停止出声，此时，需要重新插拔音频线才能激活声音的输出。


解决方法
==========

将 Windows 的声音设置到足够大，比如 70 以上，通过调节 Bose SoundLink 上的 +/- 键来调节音量。
